
A simple test command

Usage: test-cli simple [OPTIONS]

Options

  -D<name>=<value>    Define a system property
  --verbose           Produce verbose output
  --debug             Produce debug output
  --error             Print error stack traces
  --plain             Do not use color or styles in output
  --args-file         Path to a file with arguments for Helidon CLI tool
  --props-file        Path to a properties file with user inputs for Helidon CLI tool
  --foo               Foo option
  --bar               Bar option
